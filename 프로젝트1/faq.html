❓ 자주 묻는 질문 (FAQ)

🔧 서비스 관련

Q1. 이 서비스는 무료인가요?

네, 완전 무료입니다. 회원가입도 필요 없고, 모든 기능을 제한 없이 사용하실 수 있습니다. 서비스 운영을 위한 최소한의 광고(Google AdSense)만 표시되며, 이는 서버 유지비와 개발비를 충당하는 데 사용됩니다.

Q2. 개인정보를 수집하나요?

아니요. 우리 서비스는 회원가입이 없으며, 이름, 이메일, 전화번호 등 어떠한 개인정보도 수집하지 않습니다. 쿠키는 Google AdSense가 광고 타겟팅을 위해 사용할 수 있으나, 이는 익명 데이터이며 브라우저 설정에서 쿠키를 차단하실 수 있습니다.

자세한 내용은 개인정보처리방침을 확인하세요.

Q3. 모바일에서도 사용할 수 있나요?

네, 모든 기능이 PC, 태블릿, 스마트폰에서 완벽하게 작동하도록 반응형으로 제작되었습니다. 언제 어디서나 편리하게 이용하세요!

Q4. 동행복권 공식 서비스인가요?

아니요. 본 서비스는 동행복권과 공식적인 제휴 관계가 없는 독립 서비스입니다. 동행복권 공식 API와 웹사이트에서 제공하는 공개 데이터를 활용하여 통계 분석 서비스를 제공합니다.

복권 구매는 반드시 동행복권 공식 사이트나 가까운 복권 판매점을 이용하세요.

🤖 AI 추천 관련

Q5. AI 추천 번호는 어떻게 만들어지나요?

우리의 S-Score(Statistical Score) 알고리즘은 다음 요소를 종합적으로 분석합니다:

  • 출현 빈도: 역대 당첨번호에서 각 숫자가 나온 횟수
  • 최근 트렌드: 최근 회차에 더 높은 가중치 부여
  • 간격 분석: 번호 사이의 평균 간격과 표준편차
  • 조합 분석: 함께 나온 번호들의 상관관계

이러한 통계적 분석을 통해 "무작위"보다는 조금 더 "의미 있는" 번호를 추천합니다.

Q6. AI 추천 번호를 사용하면 당첨 확률이 높아지나요?

아니요. 복권은 완전히 무작위로 추첨되기 때문에, 어떤 번호를 선택하든 당첨 확률은 동일합니다.

AI 추천은 통계적 패턴을 바탕으로 "재미있는" 선택지를 제공하는 것이며, 당첨을 보장하거나 확률을 높이는 것이 아닙니다. 오락과 의사결정의 즐거움을 더하는 도구로 활용하세요.

Q7. 데이터는 어디서 가져오나요?

모든 당첨번호 데이터는 동행복권 공식 API 및 네이버 검색 결과에서 수집합니다. (공식 사이트가 일시적으로 차단될 경우 네이버를 보조 데이터 소스로 사용합니다)

매주 추첨 후 GitHub Actions를 통해 자동으로 최신 데이터를 업데이트하며, 1회차부터 최신 회차까지의 전체 데이터를 보유하고 있습니다.

💰 당첨 및 수령 관련

Q8. 복권에 당첨되면 어떻게 해야 하나요?

1단계: 복권 확인 및 보관

  • 복권 뒷면에 즉시 서명 (도난 방지)
  • 복권을 훼손하지 않도록 안전한 곳에 보관
  • 당첨 사실을 함부로 공개하지 않기

2단계: 수령 장소 방문

  • 1등: 농협은행 본점 (서울 중구, 예약 필수)
  • 2~3등: 전국 농협은행 지점
  • 4~5등: 복권 판매점

3단계: 필요 서류 지참

  • 당첨 복권 원본
  • 신분증 (주민등록증, 운전면허증, 여권)

자세한 내용은 복권 가이드를 참고하세요.

Q9. 당첨금에서 세금은 얼마나 떼나요?

당첨금액에 따라 세율이 다릅니다:

  • 5만원 이하: 세금 없음
  • 3억원 이하: 22% (소득세 20% + 지방소득세 2%)
  • 3억원 초과: 33% (소득세 30% + 지방소득세 3%)

예를 들어 10억원에 당첨되면 세금 3억 3천만원을 제외한 6억 7천만원을 실제로 받게 됩니다.

본 사이트의 세금 계산기를 사용하면 실수령액을 쉽게 계산할 수 있습니다!

Q10. 복권을 잃어버렸는데 당첨금을 받을 수 있나요?

불가능합니다. 복권은 무기명 유가증권이므로, 복권 원본이 없으면 당첨금을 받을 수 없습니다.

따라서 복권을 구매한 후에는:

  • 뒷면에 즉시 서명하기
  • 사진 찍어두기 (추가 안전 장치)
  • 안전한 곳에 보관하기
  • 절대 물에 젖거나 훼손되지 않도록 주의

🛠️ 기술 및 문제 해결

Q11. 최신 당첨번호가 업데이트되지 않았어요

당첨번호는 매주 추첨 후 자동으로 업데이트됩니다:

  • 로또 6/45: 매주 토요일 밤 추첨 후 1시간 이내
  • 연금복권 720+: 매주 목요일 저녁 추첨 후 1시간 이내

만약 업데이트가 지연된다면:

  • 브라우저 캐시 삭제 후 새로고침 (Ctrl + Shift + R)
  • 공식 사이트 점검 또는 네트워크 문제일 수 있음
  • 24시간 이상 지연 시 GitHub Issues로 문의
Q12. 페이지가 제대로 표시되지 않아요

다음 방법을 시도해보세요:

  • 캐시 삭제: Ctrl + Shift + R (강력 새로고침)
  • 브라우저 업데이트: 최신 버전의 Chrome, Edge, Safari 사용 권장
  • 애드블록 확인: 일부 광고 차단 프로그램이 페이지를 방해할 수 있음
  • JavaScript 활성화: 브라우저 설정에서 JavaScript가 켜져 있는지 확인

그래도 해결되지 않으면 GitHub Issues에 브라우저 종류, 기기 정보와 함께 문의해주세요.

Q13. 버그를 발견했어요. 어디에 신고하나요?

버그 리포트는 언제나 환영합니다! 🙏

GitHub Issues 에 다음 정보와 함께 등록해주세요:

  • 발생한 문제 상세 설명
  • 재현 방법 (어떤 순서로 조작했는지)
  • 사용 중인 브라우저 및 기기 정보
  • 가능하다면 스크린샷

빠른 시일 내에 확인하고 수정하겠습니다!

💬 기타

Q14. 새로운 기능을 제안하고 싶어요

기능 제안은 항상 환영합니다! 🎉

GitHub Issues 에 "[기능 제안]" 태그와 함께 다음 내용을 적어주세요:

  • 왜 필요한지 (어떤 문제를 해결하는지)
  • 어떻게 작동하면 좋을지

모든 제안을 검토하며, 유용하다고 판단되면 우선순위에 따라 개발하겠습니다.
